## About
基于 Python3 读取pdf文件内容, 支持多页面读取并存储为csv文件

## Feature

* 引用 pdfplumber 库, 读取pdf内容
* 基于土地登記謄本pdf文件, 找到定制字段对应内容, 且支持多页面读取
* 存储数据为csv文件

## Requires
Python 3.11.0  
pdfplumber 0.7.6  
pandas 1.5.2  

## Usage
```
python main.py
```
